This repository contains the embedded firmware for an IoT-based inventory monitoring system developed using ESP32.

It handles:
• RFID authentication (MFRC522 module)
• QR code scanning logic integration
• HTTP-based communication with external application
• Data logging via SD card module
• Multi-client support via ESP32 access point

This firmware was part of a larger full-system thesis project that includes a mobile application and inventory database system.

PS: Android application was developed separately by a team member and is not included in this repository.
